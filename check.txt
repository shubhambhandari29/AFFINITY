React-based frontend integrated with a Python FastAPI backend through secure REST APIs for SAC and Affinity workflows.
Backend is built in Python 3.11 using FastAPI, with modular service architecture and Pydantic validation for structured, reliable APIs.
Hosted on Azure Linux Web App with Azure DevOps CI/CD pipeline for automated build and deployment.
Integrated with SQL Server/Azure SQL using ODBC and Azure AD-compatible database authentication for enterprise data access.
Enterprise authentication is supported through F5/SSO at the access layer, with the API enforcing secure JWT session management using HTTP-only cookies and refresh-token flow.
